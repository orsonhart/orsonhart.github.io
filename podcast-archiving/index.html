Archiving Hamish and Andy

September 17, 2023

As a digital minimalist, I often want to spend time disconnected. To limit the amount of time I spend using a smartphone, I like to listen to music, audiobooks, and podcasts on my 2009 iPod Classic.

One challenge is that some media is resistant to being accessed offline. This includes my absolute favourite - the Hamish and Andy podcast - which has its entire back catalogue available on Libsyn available with a premium subscription. This post will explain how to download these premium podcasts for archiving or offline listening.

Note: These instructions are designed to work on Windows.

Tools Required

  • A premium Hamish and Andy subscription
  • yt-dlp

Steps

  1. Create a folder for your downloads and put yt-dlp.exe here.
  2. Login to Libsyn and navigate to the page at handa.libsyn.com listing the podcasts you want to download. Scroll to the bottom to load them all.
  3. Open the developer console and use console.log([...document.querySelectorAll("h2 > a.read_more")].map(x => x.href).join("\n")) to get a list of urls. Save these in your folder as urls.txt.
  4. Open Terminal and run the command yt-dlp --cookies-from-browser BROWSER --batch-file urls.txt -o "/YYYY/MM/%(upload_date>%Y-%m-%d)s %(title)s.%(ext)s" --parse-metadata "title:- %(title)s". Make sure to replace BROWSER with the name of your browser e.g. chrome and /YYYY/MM with the appropriate year and month, e.g. /2013/02.

Note: The --parse-metadata flag stops “Hamish & Andy” from duplicating in the filename if the podcast titles incude “Hamish & Andy -”, and can be removed if appropriate.

Let yt-dlp run and all the podcast files should be downloaded to the specified /YYYY/MM folder. A screenshot of the podcasts from February 2013 is below.
